# **App Name**: Connectonic.com

## Core Features:

- Song Browsing: Browse and listen to a curated selection of songs from emerging artists.
- Playlist Creation: Create personalized playlists.
- AI Song Recommendations: Use a generative AI tool that analyzes listening habits and suggest new songs.
- Link music platform: Link to music platform account
- Artist event Display: Display upcoming gigs by platform artists, based on location.

## Style Guidelines:

- Primary color: A vibrant electric blue (#7DF9FF), symbolizing energy, connection, and the modern sound of emerging artists.
- Background color: A deep, desaturated grayish blue (#293A40) provides contrast, grounding the electric blue and focusing user attention.
- Accent color: A saturated purple (#BE29EC) adds depth, signaling creativity and new possibilities in music.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, computerized aesthetic that reflects innovation in music.
- Body font: 'Inter' (sans-serif) for readability and a clean, objective presentation.
- Use clean, minimalist icons, emphasizing clarity and modern style.
- Employ subtle animations for song transitions and loading sequences to create a smooth user experience.
