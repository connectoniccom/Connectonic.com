# **App Name**: Connectonic.com

## Core Features:

- AI Music Generator: AI-powered song generator that suggests melody ideas and chord progressions based on user input such as preferred genre and tempo; this generator acts as a musical tool.
- User Input Interface: Simple, easy-to-use interface for inputting music preferences.
- Music Display: Display of AI-generated musical snippets (melody and chords).
- Playback Controls: Playback feature for previewing the generated music.
- MIDI Export: Option to export generated ideas in a standard MIDI format.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to evoke a sense of creativity and depth.
- Background color: Light lavender (#E6E6FA) for a calming and uncluttered feel.
- Accent color: Vibrant magenta (#FF00FF) to highlight interactive elements and AI suggestions.
- Body and headline font: 'Inter' sans-serif for a clean, modern interface.
- Minimalist icons that represent musical elements, like notes, chords, and tempo adjustments.
- Clean and structured layout with intuitive placement of user input fields, music display area, and playback controls.
- Subtle transitions and animations for visual feedback when generating or exporting music.
